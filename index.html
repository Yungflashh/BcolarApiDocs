BColar API Documentation

POST - Register Artisan

URL: https://backend-bcolar.onrender.com/api/auth/register

Request:

'https://backend-bcolar.onrender.com/api/auth/register' \
--header 'Accept: application/json' \
--form 'name="uregu"' \
--form 'email="mirabelureg1u@gmail.com"' \
--form 'phone="09012130381"' \
--form 'password="@Uregu1234"' \
--form 'regionLGA="lagos,Nigeria"' \
--form 'user_type="artisan"' \
--form 'state="benue"' \
--form 'officeAddress="49, adebiyi"' \
--form 'dateOfBirth="4th june"' \
--form 'gender="female"' \
--form 'idCard= an image url or file' \
--form 'skill="baker"'

Response:

{
  "message": "Registration successful. Check your email for OTP."
}

POST - Verify OTP

URL: https://backend-bcolar.onrender.com/api/otp/verify-otp

Example:

{
  "email": "mirabelureg1u@gmail.com",
  "otp":"291734",
  "type":"register"
}
